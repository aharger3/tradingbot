## SECTION 6: CONSOLIDATED SUMMARY

### Rules — all sources agree:
1. Re-entry only after stop-out (not premature exit)
2. Exact same setup: break, retest, hammer, stop, target
3. A+ quality only
4. Same thesis/market regime (trending must still be trending)
5. Same key level
6. Wait 5-20 minutes before re-entering
7. Disqualifiers: market regime change, broke structure other way, no displacement, multiple touches, weak confluences

### Sizing — contradictory evidence:

|--------|------|----------|
| Accelerator | SAME size (1x) | "If I'm risking $1,000… I'm gonna risk $1,000 on the second" |
| YouTube (raw) | Can size up 2x-3x | "Doubled or tripled my size on the second one" |
| YouTube (raw) | Technically can size up | "You could size up on the second one" |

### Key open questions:
1. **Is accelerator "same size" the taught rule and YouTube "size up" is the exception?** The accelerator is the course; YouTube is free content. The course might teach conservative sizing, while in practice Scarface sizes up on A+ setups he's confident in.
2. **Does "same stop same targets" in the accelerator refer to the setup criteria (stop placement, target level) or the dollar risk?** Context suggests it means same setup parameters, not necessarily same contract count. But the $1,000 quote is about dollar risk.
3. **Tony's -$2,900 → +$6,000 is 2.07x sizing.** This is concrete evidence of sizing up in practice. But is this the "right way" or is this his personal risk tolerance after years of experience?
4. **The 84% rule is NOT in the accelerator course** (confirmed by VesUpLmHCcw). It's taught in YouTube videos. This explains why the accelerator says "same size" — it may be a simplified version for students.
